# 家計簿分割入力支援アプリ

英語名: Expense Split Manager

GitHubリポジトリ: `expense-split-manager`

マネーフォワードへ高額支出や任意の支出を数ヶ月に分けて入力するための、分割入力専用PWAアプリです。
実際に支払った「実支出」と、家計簿上で月ごとに入力する「配分入力」を分けて管理します。

## 主な機能

- 分割入力専用の商品登録
  - カテゴリ
  - 日付
  - 内容
  - 支出元
  - メモ
  - 金額（税込）
  - 分割月数
  - 開始月
- マネーフォワード風の大分類・小分類カテゴリ
- カテゴリ管理
  - 大分類は初期一覧を固定
  - 小分類の追加、名前変更、削除
- 今月の入力予定表示
- 月別予定一覧
- 入力済み / 未入力の切り替え
- 登録済み商品の一覧、編集、削除
- JSONバックアップ / 復元
- 商品一覧CSV / 分割予定CSVのエクスポート
- GitHub Pages / PWA対応

## データの考え方

- 実支出: 実際に日付時点で支払った金額
- 配分入力: 家計簿上で月ごとに割り当てる金額

家計簿上の入力額は税込金額を基準にし、端数は最終月で調整します。

## データ保存

データはブラウザの `localStorage` に保存されます。
外部サーバー、ログイン、クラウド同期は使用していません。

`localStorage` 保存のため、以下の場合にデータが消える可能性があります。

- ブラウザを変更した場合
- 別の端末で開いた場合
- ブラウザのキャッシュやサイトデータを削除した場合
- プライベートブラウズなど保存が残らない環境で使った場合

重要なデータは、データ管理タブからJSONバックアップしてください。

## 起動方法

依存関係をインストールします。

```bash
npm install
```

開発サーバーを起動します。

```bash
npm run dev
```

ブラウザで以下を開きます。

```text
http://localhost:5173/expense-split-manager/
```

スマホ実機から確認する場合は、同じWi-Fi内のPCでホストを指定して起動します。

```bash
npm run dev -- --host 0.0.0.0
```

PCのローカルIPアドレスを確認し、スマホのブラウザで以下のようにアクセスします。

```text
http://PCのIPアドレス:5173/expense-split-manager/
```
